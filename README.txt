# Pokemon-Planet-Autofarm

## YOU WILL NEED:
---------------------------------
PYTHON 3.10 OR LATER


STEP 1: Run command prompt as an administrator on your pc (Make sure you have python installed!) 
STEP 2: Type "pip install pyautogui" (Some people may need to run "pip3 install pyautogui")
STEP 3: Open autofarm.py

----------------------------------

MAKE SURE YOUR CHARACTER IS IN A GRASSY AREA WHERE YOU CAN CATCH POKEMON BEFORE RUNNING, ENJOY!
